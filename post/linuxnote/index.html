一、vim

  • 插入模式

    按下i,o,a,r等任何一个字母后可以进入编辑模式，一般按i即可

  • 命令行模式

    输入 esc 再输入 : 在这个模式中可以提供相关的指令，完成读取、存盘、替换、离开vim等操作

  • 快捷键

    • yy 拷贝 p 粘贴
    • dd 删除

二、开机、重启、注销

开机： shutdown -h now 立即关机
	  shutdown -h  1  一分钟后关机

重启： shutdown -r row 立即重启
	  reboot
	  
注销： logout

三、用户管理

3.1 添加和删除用户

添加用户： useradd + 用户名

修改用户密码： passwd + 用户名

删除用户： userdel + 用户名

3.2 用户组

新增组： groupadd + 组名

删除组： groupdel + 组名

添加用户时指定组： useradd -g + 用户组 + 用户名

修改用户的组： usermod -g + 用户组 + 用户名

四、文件目录指定

4.1 pwd ls cd

pwd 显示当前工作目录的绝对路径

ls 查看当前目录的内容信息
	-a 显示全部包括隐藏
	-l 以列表的方式显示信息

cd 进入目录

4.2 mkdir rmdir touch

mkdir 创建目录 
 -p 创建多级目录
 
rmdir 删除空目录

touch 创建空文件

4.3 cp rm

cp 拷贝文件到指定目录
 cp [-r] source dest
 -r 递归复制整个文件夹

rm 删除文件或目录
 -r 递归删除整个文件夹
 -f 强制删除不提示

4.4 mv cat

mv 移除文件与目录或重命名
	mv oidFile newFile 在同个目录下mv是重命名
	mv /xxx/xx /xx     mv到不同目录就是移动
	
cat 查看文件内容
	-n 显示行号

4.5 echo head tail > »

echo 输出内容到控制台

head 显示文件开头的部分内容
	head -n 5 文件 查看文件头5行内容，5可以换成任意行数

tail 显示文件尾部的部分内容
	 tail -n 5 文件 查看文件尾5行内容，5可以换成任意行数

4.6 时间日期指令

date  显示当前时间
date + %Y 显示当前年
	 + %m 显示当前月
	 + %d 显示当前日
	 + %Y-%m-%d %H:%M:%S 显示年月日时分秒

五、查找指令

5.1 find which

find [搜索范围] [选项]
	-name	按照指定文件名查找
	-user	按照指定用户名查看所以文件
	-size	按照指定文件大小查找
	
which 可以查看某个指令在哪个目录下，比如 ls 指令在哪个目录下
	which ls

5.2 grep 管道符号|

grep 过滤查找，管道符 ‘|’ ，表示将前一个命令的处理结果输出传递给后面的命令处理

grep [选项] 查找内容 源文件
	-n 显示匹配行及行号
	-i 忽略字母大小写

六、压缩和解压

6.1 gzip/gunzip zip/unzip

gzip / zip 压缩文件
gunzip / unzip 解压文件

6.2 tar

打包：
tar -czvf 打包文件名 .tar.gz 源文件或目录
tar -cjvf 打包文件名 .tar.bz2 源文件或目录
tar -cJvf 打包文件名 .tar.xz 源文件或目录

解包：
tar -xzvf 解包文件名 .tar.gz [-C 解压目录]
tar -xjvf 解包文件名 .tar.bz2 [-C 解压目录]
tar -xJvf 解包文件名 .tar.xz [-C 解压目录]

七、组管理和权限管理

7.1 所有者和所在组

一般为文件的创建者，谁创建了该文件，就自然成为该文件的所有者

当用户创建一个文件后，这个文件的所在组就是该用户所在的组

7.2 修改所有者和所在组

chown newowner 文件/目录 改变所有者
chown newowenr:newgroup 文件/目录 改变所有者和所在组
 -R 如果是目录，递归生效
 
chgrp newgroup 文件/目录 改变所在组

7.2 rwx权限

[r] 代表可读：可以读取查看
[w] 代表可写：可以修改但不代表可以删除文件
[x] 代表可执行：可以被执行

7.3 修改权限

  • 方法一：+、-、= 变更权限

    u：所有者 g：所在组 o：其他人 a：所有人

    chmod u=rwx,g=rx,o=x 文件/目录
chmod o+w 文件/目录
chmod a-x 文件/目录

  • 方法二：通过数字变更权限

    r=4 w=2 x=1 rwx=4+2+1=7

    chmod 777 文件/目录

八、进程管理

8.1 ps指令

ps 用来查看目前系统中正在执行的进程以及它们的状态
	pa -aux|grep xxx[服务名] 查看指定服务进程状态

8.2 终止进程

kill [选项] 进程号
	-9 表示强迫进程立即停止
killall 进程名

8.3 systemctl指令

  • systemctl 管理指令

    systemctl [start/stop/restart/status] 服务名

  • systemctl 设置服务的自启动状态

    systemctl list-unit-files [|grep 服务名] （查看服务开机启动状态，grep可以进行过滤）
systemctl enable 服务名 （设置服务开机启动）
systemctl disable 服务名 （关闭服务开机启动）

8.4 firewall指令

打开端口：firewall-cmd --permanent --add-port=端口号/协议
关闭端口：firewall-cmd --permanent --remove-port=端口号/协议
重新载入才能生效：firewall-cmd --query-port=端口号/协议

8.5 top动态监控进程

top与ps命令相似

top与ps最大的不同是，top在执行时间可以更新正在运行的进程

top [选项]
	-d（秒数） 指定top命令每隔几秒更新，默认3秒
	-i 使top不显示任何闲置或者死进程
	-p 通过进程id仅监控某个进程状态
